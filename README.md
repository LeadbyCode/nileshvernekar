# Nilesh Vernekar - Portfolio Website

A modern, responsive portfolio website showcasing my work as an iOS Developer. Built with HTML, CSS, and JavaScript with smooth animations and a dark mode toggle.

## Features

- **Modern Design**: Clean, minimalist aesthetic inspired by top portfolio websites
- **Dark Mode**: Toggle between light and dark themes
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: Fade-in effects, parallax scrolling, and interactive elements
- **Performance Optimized**: Fast loading times and smooth scrolling
- **Interactive Elements**: Magnetic buttons, hover effects, and animated counters

## Sections

1. **Hero Section**: Introduction with key statistics
2. **About**: Background and expertise overview
3. **Experience**: Timeline of professional experience
4. **Projects**: Featured applications and achievements
5. **Skills**: Technical expertise and tools
6. **Contact**: Get in touch form and contact information

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6+)
- Google Fonts (Outfit, Instrument Serif)

## Local Development

1. Clone this repository
2. Open `index.html` in your browser
3. No build process required!

## Deployment Options

### Option 1: GitHub Pages (Recommended)

1. Create a new repository on GitHub
2. Push your code to the repository
3. Go to Settings > Pages
4. Select the main branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify

1. Sign up for a free account at [Netlify](https://netlify.com)
2. Drag and drop your project folder
3. Your site will be live instantly with a custom URL

### Option 3: Vercel

1. Sign up for a free account at [Vercel](https://vercel.com)
2. Import your GitHub repository
3. Deploy with one click

## Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --color-primary: #667eea;
    --color-accent: #764ba2;
    /* Add your custom colors */
}
```

### Adding Content

Update the content in `index.html`:
- Projects: Add new `.project-card` elements
- Experience: Add new `.timeline-item` elements
- Skills: Add new skill tags or categories

### Contact Form

The contact form currently shows an alert. To make it functional:
1. Set up a backend API endpoint
2. Update the form handler in `script.js`
3. Or use a service like Formspree, Netlify Forms, or EmailJS
