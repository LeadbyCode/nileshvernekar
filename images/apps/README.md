# App Screenshots Directory

## Structure
This directory contains screenshots for each app featured in the portfolio.

## Instructions
Please save the app screenshots you shared into the appropriate folders:

### Folder Mapping:
- `verizon-protect/` - 5 screenshots (Expert support, Wi-Fi scan, Security assessment, etc.)
- `starbucks/` - 3 screenshots (Find Store, Rewards, Scan)
- `tim-hortons/` - 4 screenshots (Checkout, Offers, Rewards, Welcome)
- `viya/` - 4 screenshots (Golf, Introducing, Access, Earn & Save)
- `club-apparel/` - 5 screenshots (Gift Cards, Products, Invoice, Offers, Rewards)
- `hear-com/` - 5 screenshots (German UI, Adjust, Simple, Control, My Mode)

### Naming Convention:
- Use `screenshot-1.jpg`, `screenshot-2.jpg`, etc.
- Or use descriptive names that match the JavaScript configuration in script.js
